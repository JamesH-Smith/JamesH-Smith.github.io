Project Overview

This project focused on how accurate and efficient a custom numercial sizing method can design and converge on a simple and efficient flying wing. This is because speed and accuracy is important in producing and testing UAV designs. Flying-wing configurations exhibit strong coupling between geometry, centre of gravity, trim, and static stability, making traditional spreadsheet-based sizing inefficient and error-prone.

A Newton–Raphson–based solver was developed in MATLAB to simultaneously solve for wing geometry and control trim directly from physical constraints. The resulting aircraft configuration was independently validated using XFLR5, confirming aerodynamic efficiency and dynamic stability prior to CAD and manufacturing.

Coupled Geometry and Trim Solver (Newton–Raphson)

Solver Formulation

  • Unknowns: root chord, wing sweep angle, elevon trim deflection
  • Constraints: lift = weight, Cm = 0, positive static margin (~10%)
  • Method: Newton–Raphson with damping and numerical Jacobian
Initial Guess → Geometry & CG Model → Aerodynamic Model → Trim & Stability Constraints → Newton Update → Converged Design

This formulation allows the solver to converge rapidly to a physically meaningful configuration, avoiding over-reflexed or marginally stable solutions common in manual design workflows.

Solver Output: Final Trimmed Configuration

  • Root chord: 21.8 cm
  • Span: 97.5 cm
  • Sweep angle: 13.4°
  • Elevon trim: −3°
  • Static margin: ~8–10%

This configuration represents a balance between aerodynamic efficiency, pitch stability, and control authority suitable for low-Re UAV operation.

Aerodynamic & Dynamic Validation (XFLR5)

❮ ❯

Static Performance

XFLR5 analysis confirmed peak lift-to-drag ratios of approximately 27–30, with the trimmed operating point lying close to maximum efficiency. This validates the solver’s ability to produce an efficient, non-over-reflexed flying-wing configuration.

Dynamic Stability

Dynamic analysis showed a weakly damped phugoid mode, typical of flying-wing aircraft, and a well-damped short-period mode (ζ ≈ 0.5), indicating a stable and controllable pitch response near cruise conditions.

Manufacturing

The solver-derived geometry was translated into CAD and manufactured using XPS foam with fibreglass skin reinforcement. Component placement followed the CG location predicted by the numerical model, minimising the need for ballast or post-build trimming.
