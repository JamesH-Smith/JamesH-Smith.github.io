Project Overview

This project demonstrates the complete mechanical engineering workflow transitioning from a conceptual 3D CAD assembly to a high precision physical product using manual machining techniques while meeting specific tolerances. The objective was to manufacture a robust, interlocking toolbox based on strict engineering drawings, requiring an understanding of material properties, tool geometry, and tolerances.

  • CAD/Fusion360
  • Lathe
  • Vertical Milling
  • Drill press
  • Benchwork
  • Metrology

UAV Preliminary Design - Rendered

Constraint Diagram: Design Choice

Chosen Thrust to Weight Ratio: W/S ≈ 40, T/W ≈ 1.0, Reynold's Number ≈ 180,000 ≈ 15m/s​ V,Stall = 7.71m/s

Manual Milling Operations:: Using a Vertical Milling Machine, I squared the stock to within a 5mm tolerance. Key skills included using a Dial Test Indicator to "tram" the mill head and an Edge Finder to establish a precise (0,0) datum for all subsequent pocketing and drilling operations.

UAV Main Wing and Tail Sizing

Aircraft was sized using wing loading to calculate wing area, where then minimisng induced drag influenced the choice of the aspect ratio and therefore the rest of the UAV. Iterations and Aerodynamic performance was calculated using XFLR5 and hand written equations.

Tooling Selection:Choosing appropriate Carbide vs. HSS (High-Speed Steel) cutters based on the material's surface feet per minute (SFM) requirements.
