Off-Board Object Tracking UAV

← Back to Portfolio

Project Overview

This project focuses on the development of an off-board "click-to-track" system for a composite fixed-wing UAV. By offloading heavy computation to a ground station laptop, the system can perform complex image processing without taxing the aircraft's onboard power.

  • OpenCV
  • Python
  • ESP32
  • Kalman Filtering
  • MAVLink

Technical Breakdown

The Tracking Pipeline: The system utilizes an OpenCV-based tracker running on a ground-station. Once a target is selected via the GUI, coordinates are processed through a Kalman Filter implemented on an ESP32 to smooth out noise and predict target motion.

Integration: Data is transmitted via a telemetry link to the flight controller, commanding the UAV to maintain a specific offset from the tracked object for persistent ISR (Intelligence, Surveillance, and Reconnaissance).
